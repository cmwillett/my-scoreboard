- [ ] Notes for games
- [ ] Should show a "Last WC Update" message to show the last time it was refreshed
- [ ] Should have a searchable dropdown with all the teams in the WC in it to add/follow
- [ ] There should be 3 sections:  Live, Upcoming, Final, and Schedule - Live would show any games that are live with any teams that I've followed/favorited.  Upcoming would show any games that have a team that I've followed/favorited that aren't in a currently live game - it should show their next game, not all games scheduled.   Final would show any games that have a team that I've followed/favorited that have completed, sorted by most recent to the oldest.  Schedule should just show the remaining schedule for All teams regardless of favorites and followed teams, in order of next game to last game.
